AEGIS (ระบบเกราะป้องกันโฆษณารบกวนและภัยพนันออนไลน์อัจฉริยะ) 
ที่มาและความส าคัญ 
ในยุคดิจิทัลปัจจุบัน สมาร์ตโฟนได้กลายเป็นปัจจัยที่ 5 ในการด ารงชีวิต โดยพฤติกรรมการเสพข้อมูล
ข่าวสารได้เปลี่ยนไปจากการเข้าชมผ่านเว็บเบราว์เซอร์โดยตรง มาเป็นการกดลิงก์ผ่านแอปพลิเคชันโซเชียลมีเดีย
ยอดนิยม เช่น Facebook, LINE หรือ X ซึ่งจะเปิดหน้าเว็บไซต์ผ่าน "เบราว์เซอร์ภายในแอป" (Internal 
Browser/WebView) ปัญหาส าคัญ (Pain Point) ที่สร้างความร าคาญให้กับผู้ใช้งานมากที่สุดคือ "โฆษณารบกวน" 
(Intrusive Ads) ที่มักจะเด้งขึ้นมาปิดบังเนื้อหาข่าว ท าให้เสียสมาธิในการอ่าน และในหลายครั้งโฆษณาเหล่านี้ยัง
ท าการเปลี่ยนหน้าเว็บอัตโนมัติ (Auto-redirect) ไปยังแอปพลิเคชันร้านค้าออนไลน์หรือเว็บไซต์อื่นโดยที่ผู้ใช้ไม่
ต้องการ ซึ่งท าลายประสบการณ์การใช้งาน (User Experience) อย่างรุนแรง นอกจากนี้ โฆษณาเหล่านี้ยังเป็น
ช่องทางหลักที่มิจฉาชีพใช้แฝงภัยคุกคาม (Malvertising) เพื่อน าทางผู้ใช้ไปสู่เว็บไซต์อันตราย  
ปัญหาที่ทวีความรุนแรงและส่งผลกระทบต่อสังคมในวงกว้างคือการแพร่ระบาดของ 
"เว็บไซต์การพนัน
ออนไลน์" ที่มุ่งเป้าไปยังเด็กและเยาวชน โดยอาศัยช่องโหว่ของเทคโนโลยีบล็อกโฆษณาทั่วไป เว็บพนันเหล่านี้มักใช้
เทคนิคการฝังโฆษณาแบนเนอร์ (First-party Ads) ปะปนไปกับเนื้อหาของเว็บไซต์กลุ่มเสี่ยง เช่น เว็บดูหนัง
ออนไลน์หรือเว็บการ์ตูนละเมิดลิขสิทธิ์ ซึ่งระบบบล็อกระดับเครือข่าย (DNS Blocking) ไม่สามารถแยกแยะและปิด
กั้นได้เนื่องจากโฆษณาถูกส่งมาจากเซิร์ฟเวอร์เดียวกับเนื้อหาหลัก อีกทั้งเว็บพนันยังมีการเปลี่ยนชื่อโดเมนเว็บไซต์
หนีการตรวจจับเป็นรายวัน (Domain Hopping) ท าให้ฐานข้อมูลบัญชีด า (Blacklist) แบบดั้งเดิมอัปเดตตามไม่ทัน 
ส่งผลให้เยาวชนยังคงมองเห็นสื่อชักจูงและเข้าถึงการพนันได้ง่ายดาย 
จากปัญหาดังกล่าว ผู้พัฒนาจึงมีแนวคิดที่จะพัฒนา "AEGIS" (อีจิส) แอปพลิเคชันเกราะป้องกันภัยดิจิทัล
ที่ท างานแบบ "Defense in Depth" โดยระบบจะท างานเชิงรุก 2 ระดับ เพื่อแก้ปัญหา Pain Points ข้างต้น
โดยเฉพาะ ระดับที่ 1 คือ การกรองระดับเครือข่าย (Network Shield) เพื่อปิดกั้นโฆษณาและการ Redirect ใน
ทุกแอปพลิเคชัน และระดับที่ 2 คือ การวิเคราะห์และลบองค์ประกอบหน้าจอ (DOM Sanitization) เพื่อก าจัด
แบนเนอร์พนันที่ฝังเนียนมากับหน้าเว็บไซต์ ส าหรับการท างานของระบบ (Business Flow) เริ่มต้นเมื่อผู้ใช้งานเปิด
แอปพลิเคชัน ระบบจะสร้าง Local VPN Service ท างานเบื้องหลังเพื่อดักจับและกรอง DNS Requests ของ
โฆษณาและเว็บพนันทันที (แก้ปัญหาโฆษณาใน Facebook/LINE) หากผู้ใช้งานเข้าสู่เว็บไซต์กลุ่มเสี่ยง ระบบจะท า
การ Inject Script เพื่อวิเคราะห์โครงสร้าง HTML (DOM Heuristics) และสั่ง "ลบ" (Remove) องค์ประกอบที่
เป็นโฆษณาพนันออกจากหน้าจอโดยอัตโนมัติ พร้อมทั้งมีระบบ Heartbeat Watchdog คอยตรวจสอบสถานะ
แอปพลิเคชันเพื่อป้องกันการถูกปิดหรือถอนการติดตั้งโดยบุตรหลาน 
คุณสมบัติส าคัญของระบบ (Features) 
1. System-wide Network Shield: ระบบปิดกั้นโฆษณาและตัวติดตามพฤติกรรมระดับเครือข่าย โดยใช้
เทคนิค DNS Sinkholing ผ่าน Android VPN Service ท าให้สามารถยับยั้งการโหลดโฆษณาและการ 
Redirect ที่สร้างความร าคาญได้ครอบคลุมทุกแอปพลิเคชัน รวมถึง Internal Browser ใน Facebook 
และ LINE 
2. Intelligent DOM Sanitizer: ระบบวิเคราะห์และลบโฆษณาแฝงบนหน้าเว็บแบบอัตโนมัติ 
(Automated Banner Removal) โดยใช้อัลกอริทึมตรวจสอบโครงสร้าง HTML (DOM Heuristics) เพื่อ
ตรวจจับแบนเนอร์พนันจากพฤติกรรมของโค้ด เช่น การตรวจสอบลิงก์ปลายทาง (Target URL) หรือการ
ซ่อน iframe และสั่งลบ Element นั้นออกจากหน้าจอทันที เพื่อแก้ปัญหา First-party Ads ในเว็บหนัง
เถื่อน 
3. Strict Anti-Gambling Mode: ระบบตรวจสอบและปิดกั้นการเข้าถึงเว็บพนันเชิงรุก โดยตรวจสอบ 
URL เทียบกับฐานข้อมูลบัญชีด า และวิเคราะห์ Keyword เสี่ยงในลิงก์ เพื่อป้องกันการเข้าถึงเว็บพนันที่
เปลี่ยนโดเมนใหม่ (Domain Hopping) 
4. Anti-Tamper & Heartbeat Watchdog: ระบบป้องกันการปิดการท างานหรือถอนการติดตั้งแอป
พลิเคชัน โดยอาศัยสิทธิ์ Device Administrator และ Accessibility Service ในการป้องกันการเข้าถึง
เมนูตั้งค่า และมีระบบส่งสัญญาณชีพ (Heartbeat) ไปยัง Server เพื่อแจ้งเตือนผู้ปกครองทางไลน์หรือ
อีเมลหากแอปถูกปิดหรือขาดการติดต่อ 
ประโยชน์ที่จะได้รับ 
1. ด้านผู้ใช้งานทั่วไป: ได้รับประสบการณ์การอ่านข่าวและท่องเว็บผ่านโซเชียลมีเดียที่ลื่นไหล ไม่มีโฆษณา
รกหน้าจอหรือการ Redirect ไปยังแอปอื่นมากวนใจ 
2. ด้านเยาวชนและผู้ปกครอง: เยาวชนได้รับการปกป้องจากสื่อยั่วยุและเว็บพนันออนไลน์ที่แฝงมากับเว็บดู
หนัง ช่วยลดความเสี่ยงในการเสพติดพนันและการสูญเสียทรัพย์สิน 
3. ด้านสังคม: ช่วยลดปัญหาอาชญากรรมไซเบอร์และการพนันออนไลน์ในกลุ่มวัยรุ่น สร้างภูมิคุ้มกันทาง
ดิจิทัลให้กับสังคมไทย 
4. ด้านเทคโนโลยี: ได้นวัตกรรมแอปพลิเคชันที่ประยุกต์ใช้เทคนิค Network Security ร่วมกับ DOM 
Manipulation บนอุปกรณ์เคลื่อนที่เพื่อแก้ปัญหาโฆษณาแฝงภาษาไทยและเว็บพนันได้อย่างตรงจุด 
5. ด้านผู้พัฒนา: ได้พัฒนาทักษะการเขียนโปรแกรม Android Native (Kotlin) การจัดการ System 
Service และการวิเคราะห์โครงสร้างเว็บไซต์ (Web Scraping/Analysis) เพื่อแก้ปัญหาจริง 
เงื่อนไขของระบบ 
1. อุปกรณ์ที่รองรับต้องเป็นสมาร์ตโฟนระบบปฏิบัติการ Android เวอร์ชัน 10.0 ขึ้นไป เพื่อรองรับการ
ทำงานของ VpnService และ Accessibility API 
2. ผู้ใช้งานจำเป็นต้องกดอนุญาตสิทธิ์ (Permissions) ที่สำคัญ ได้แก่ VPN Configuration, Display over 
other apps, Device Administrator, และ Accessibility Service เพื่อให้ระบบทำงานได้อย่างสมบูรณ์ 
3. แอปพลิเคชันต้องมีการเชื่อมต่ออินเทอร์เน็ตเพื่อใช้ในการดึงข้อมูล Threat Intelligence และส่งสถานะ 
Heartbeat ไปยัง Server 
เทคโนโลยีที่คาดว่าจะใช้ หรือเกี่ยวข้อง 
1. Mobile Application Framework: Flutter (Dart) สำหรับการพัฒนาส่วนติดต่อกับผู้ใช้งาน (UI) และ 
Logic หลักของแอปพลิเคชัน 
2. Native Android Development: Kotlin สำหรับการพัฒนาระบบ Local VPN Service เพื่อกรอง
โฆษณา และ Accessibility Service สำหรับการเข้าถึงหน้าจอ 
3. Web Injection & Scripting: JavaScript Injection สำหรับเข้าไปทำงานใน WebView เพื่ออ่านค่า 
DOM Tree และลบ Element โฆษณา (DOM Manipulation) 
4. Threat Intelligence APIs: VirusTotal API v3 และ Google Safe Browsing API v4 สำหรับ
ตรวจสอบความปลอดภัยของโดเมนและ URL 
5. Backend: Supabase สำหรับเก็บข้อมูล Log การใช้งาน ฐานข้อมูล Blacklist และระบบแจ้งเตือน 
Heartbeat
